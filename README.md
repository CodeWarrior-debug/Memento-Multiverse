## Description
---
Do you love collecting TV merch? Then you need a fun and funny web store for merch that appreciates your favorite shows, their inside jokes, and transports you into their worlds. A store where you can view your purchases by TV franchise, see how much youâ€™ve invested into each collection, and have a fun animated time while doing it. Memento Multiverse will be the most fun and funny place to collect memorabilia on the web.

## Installation
---
Visit on Heroku 

https://memento-tv-multiverse.herokuapp.com/

OR

Go to Github and Clone or Fork repository into your code editor. Run npm i in the correct file to install dependencies. Run npm start to start the application.
